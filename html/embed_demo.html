embedding demo // extinfo-web

Look at the source code of this page.

To use embed.js, you will need a container (preferrably a div) with a specific id field. The ID of that div has to start with extinfo-, followed by an identifier you can choose yourself, for example myServer. This page uses: 

<div id="extinfo-demo">
</div>
The reason behind this second part is that only so can you embed the script multiple times on one page (for when you have more than one server you want to monitor on your website).
Make sure these IDs are URL-safe!

At the end of your page's <body> ... </body>, put this snippet: 

<script src="http://{{.}}/embed.js?id=demo&amp;addr=sauerleague.org&amp;port=6666">
</script>
demo is the part of your container's ID you chose yourself (your own identifier), sauerleague.org is the IP or host name of the server you want to monitor, and 6666 its port.

You'll have to style the resulting table yourself (however, feel free to copy the style from this page if you like it; formatted CSS is here).
Note the .error class: in case the browser does not support websockets, a <div> containing more information is shown instead of the table.
To target only elements injected into the DOM tree by the embed script, use something like this: 

#extinfo-demo {
	table {
    	border-collapse: collapse;
    	color: #222;
    	...
	}

	tr {
		...
	}

	...
}
